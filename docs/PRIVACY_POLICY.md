# Политика конфиденциальности приложения «ЛК Интерра»

Дата вступления в силу: 2 июля 2026 г.

Приложение «ЛК Интерра» (далее — Приложение) — личный кабинет абонента
интернет-провайдера «Интерра». Настоящая политика описывает, какие данные
обрабатывает Приложение и для чего.

## Какие данные мы обрабатываем

1. **Номер мобильного телефона.** Используется один раз для регистрации
   Приложения в биллинге провайдера по SMS-коду. Хранится на устройстве в
   защищённом хранилище системы.
2. **Токен push-уведомлений (FCM).** Передаётся на сервер уведомлений
   провайдера, чтобы присылать сообщения о балансе, работах на сети и новостях.
   Удаляется с сервера при выходе из аккаунта или отключении уведомлений.
3. **Данные личного кабинета** (баланс, тариф, ФИО, адрес). Загружаются
   напрямую с сервера биллинга провайдера (stat.interra.ru) и отображаются в
   Приложении. Последняя загруженная страница и значение баланса кэшируются на
   устройстве для работы без сети.
4. **Обезличенная статистика использования и отчёты о сбоях**
   (Firebase Analytics, Firebase Crashlytics, Google LLC). Помогают находить
   ошибки и улучшать Приложение. Не содержат содержимого личного кабинета.

## Чего мы не делаем

- Не продаём и не передаём данные третьим лицам, кроме перечисленных
  технических сервисов.
- Не собираем геолокацию, контакты, фотографии и другие данные устройства.
- Не показываем рекламу.

## Биометрия

Вход по Face ID / отпечатку выполняется средствами операционной системы.
Биометрические данные не покидают устройство и Приложению недоступны.

## Хранение и удаление

Данные регистрации хранятся на устройстве и удаляются при выходе из аккаунта.
Чтобы удалить токен уведомлений с сервера, выйдите из аккаунта или отключите
уведомления в настройках. По вопросам удаления данных обратитесь в поддержку.
