черновик, не закоммичено. отмечай сделанное.

## сначала — общее для всех трёх

- [ ] поднять push-бекенд (`push.interra.ru` + сервер) — на админе, ждём
- [ ] проверить пуши end-to-end после деплоя
- [ ] политика конфиденциальности — выложить страницей на сайте (черновик есть в `docs/PRIVACY_POLICY.md`)
- [ ] пользовательское соглашение — тоже страницей на сайте
- [ ] демо-аккаунт для модератора (лк за логином — без рабочего входа отклонят везде)
- [ ] тексты витрины — готовы в `docs/store/store-listing.md`
- [ ] иконка и скриншоты — на дизайнерах
- [ ] ответы data safety — готовы в `docs/store/data-safety.md`

## google play

- [ ] аккаунт google play console — $25 разово
- [ ] если аккаунт личный: closed testing 20 тестеров 14 дней до релиза (заложить время)
- [ ] собрать подписанный `.aab` (ключ готов, нужен android sdk или ci)
- [ ] залить .aab, заполнить листинг (тексты готовы)
- [ ] анкета data safety (ответы готовы)
- [ ] возрастной рейтинг (анкета)
- [ ] ссылка на политику конфиденциальности
- [ ] отправить на ревью

## rustore

- [ ] аккаунт rustore — обычно нужен ип/юрлицо (оформить на компанию)

- [ ] листинг + политика конфиденциальности + возрастной рейтинг

- [ ] на будущее: rustore push sdk для устройств без google-сервисов (huawei и часть рф-прошивок) — иначе там пуши не дойдут

## app store

- [ ] сборка и загрузка через xcode на mac + provisioning-профили
