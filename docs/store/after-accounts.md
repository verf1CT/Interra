# что делаю после получения аккаунтов сторов

аккаунты дают другие админы. как только на руках доступы — по каждому стору мои шаги.

## что просить у админов (доступы)

- **google play** — приглашение в play console на мой аккаунт (роль admin/release), либо готовый аккаунт с $25 оплаченным
- **apple** — доступ в apple developer / app store connect (роль admin или app manager), программа $99/год оплачена
- **rustore** — доступ в консоль rustore (аккаунт на ип/юрлицо компании)

## google play — после доступа

1. создать приложение в play console (название, язык, тип — приложение, бесплатное)
2. залить подписанный `.aab` (собираем отдельно — ключ готов)
3. заполнить листинг — тексты из `docs/store/store-listing.md`, иконка + скриншоты от дизайнеров
4. анкета **data safety** — ответы из `docs/store/data-safety.md`
5. возрастной рейтинг (анкета), категория «утилиты»
6. вставить ссылку на политику конфиденциальности
7. если аккаунт личный — запустить **closed testing** (20 тестеров, 14 дней), потом продакшн
8. отправить на ревью

## rustore — после доступа

1. создать приложение в консоли rustore
2. залить `.aab`/`.apk`
3. листинг (те же тексты), иконка, скриншоты
4. политика конфиденциальности + возрастной рейтинг
5. отправить на модерацию
6. позже — добавить rustore push sdk для устройств без google-сервисов

## app store — после доступа

1. на mac в xcode настроить signing (команда/провижининг подтянутся из аккаунта)
2. сгенерить **apns-ключ (.p8)** в apple developer и залить в firebase (иначе пуши на iphone не идут)
3. создать приложение в app store connect (bundle id `ru.interra.lkInterra`)
4. собрать и загрузить сборку через xcode / transporter
5. листинг + скриншоты под размеры iphone + privacy labels (готовы)
6. ссылка на политику конфиденциальности
7. дать демо-логин ревьюеру, отправить на ревью

## что от меня не зависит (жду других)

- бекенд `push.interra.ru` — серверные админы
- сами аккаунты — админы с доступами
- privacy + соглашение как страницы на сайте — выложить на interra.ru
- иконка/скриншоты — дизайнеры
