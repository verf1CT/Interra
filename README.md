# ЛК Интерра

Мобильный личный кабинет абонента интернет-провайдера **Интерра** (`interra.ru`,
г. Первоуральск). Работает поверх биллинга NetUP UTM5: показывает кабинет,
подтягивает баланс нативно, шлёт push-уведомления и даёт быстрые инструменты
(диагностика сети, проверка скорости, виджеты, Siri).

## Состав репозитория

| Папка     | Что это                                                                    |
|-----------|----------------------------------------------------------------------------|
| `app/`    | Мобильное приложение (Flutter, iOS + Android) — основной продукт           |
| `server/` | Бэкенд рассылки push (Node.js/Express) + веб-панель оператора              |
| `docs/`   | Заметки по настройке платформ, политика конфиденциальности, чеклист сторов |

## Как это связано

```
┌────────────────────────┐     регистрация push-токена    ┌────────────────────────┐
│  Приложение (app/)      │ ─────────────────────────────▶ │  Бэкенд (server/)       │
│  • кабинет в WebView    │                                 │  • база устройств       │
│  • нативный баланс      │ ◀──── push (FCM → APNs/Android) │  • рассылка (firebase)  │
│  • виджеты / Siri / часы│                                 │  • панель оператора     │
└────────────────────────┘                                 └────────────────────────┘
        │  прямые запросы к биллингу (HTTPS, с пиннингом)
        ▼
   stat.interra.ru/cgi-bin/utm5  (UTM5: вход по SMS-токену, страница кабинета, баланс)
```

## Суть работы

- **Вход** — по номеру телефона и SMS-коду через штатный CGI `bbb` биллинга
  UTM5: приложение один раз регистрируется и хранит 24-значный токен, по которому
  при каждом открытии получает свежую ссылку на кабинет.
- **Кабинет** показывается в WebView (`stat.interra.ru`), а баланс и номер счёта
  дополнительно **парсятся** со страницы и выводятся нативно — в шапке, виджетах,
  на часах и через Siri.
- **Уведомления** идут через Firebase Cloud Messaging: оператор из панели
  бэкенда рассылает их всем, сегменту (по тегам-категориям) или конкретному логину.
