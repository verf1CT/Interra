# ЛК Интерра

Мобильное приложение-личный кабинет для интернет-провайдера **Интерра** (`interra.ru`).
Повторяет веб-кабинет `https://stat.interra.ru/` (биллинг NetUP UTM5), сохраняет
авторизацию и присылает push-уведомления.

## Состав

| Папка     | Что это                                                                 |
|-----------|-------------------------------------------------------------------------|
| `app/`    | Мобильное приложение (Flutter, Android + iOS): WebView + авто-логин + push |
| `server/` | Бэкенд (Node.js/Express): регистрация устройств, рассылка push, админ-панель |

## Архитектура

```
┌─────────────────┐        регистрация токена         ┌──────────────────────┐
│  Приложение      │ ───────────────────────────────▶ │  Бэкенд (server/)     │
│  (Flutter)       │                                   │  - БД устройств       │
│  - WebView ЛК    │ ◀─── push (FCM/APNs) ──────────── │  - админ-рассылка     │
│  - авто-логин    │                                   │  - firebase-admin     │
└─────────────────┘                                   └──────────────────────┘
        │                                                        ▲
        │ показывает stat.interra.ru (UTM5)              админ панель │
        ▼                                                        │
  https://stat.interra.ru/cgi-bin/utm5/aaa                   оператор
```

Уведомления отправляются через **Firebase Cloud Messaging (FCM)** — на Android напрямую,
на iOS через APNs. Оператор может разослать пуш всем клиентам, сегменту или конкретному логину.
